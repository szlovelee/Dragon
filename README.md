# Dragon

## 과제 개요
- 주제: 보스 스테이지 구현
- 시도 내용
  1. 플레이어 움직임
  2. 보스 공격 패턴
  3. 무기 이펙트
- 플레이 방법 : 용가리에게 방어의 힘을 주는 비석을 무력화 한 후 용이 쓰러지면 머리를 공격한다
